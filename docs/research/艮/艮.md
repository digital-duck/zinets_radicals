## 1. PRELIMINARY ANALYSIS

### Traditional Interpretation
- **Modern meanings:**
  1. 艮 (gěn) = "meaning a person who is blunt" (original meaning)
  2. One of the Eight Trigrams (八卦): ☶ Gèn - Mountain
  3. Northeast direction
  4. Stopping, limit, boundary
- **Traditional etymology:** Eye (目) + person looking away (person-look-away 艮 gěn)
- **Kangxi status:** Radical #138
- **Usage frequency:** Rare independently (Importance: 3524)

### Character Structure
- **Oracle bone forms:** 0 attestations (NO oracle bone evidence)
- **Bronze inscriptions:** 0 attestations (NO bronze evidence)
- **Seal script:** 1 form (S05942)
- **Liushutong variants:** 4 forms (L22328, L22329, L22330, L36494)
- **Evidence quality:** WEAK for early forms (only seal script), BUT...

**CRITICAL OBSERVATION:**
- **NO oracle/bronze evidence** (like 乇)
- **BUT HUGE character network** (20 characters identified - vs. 乇's 4 characters)
- **This is OPPOSITE pattern from 乇:**
  - 乇: No early evidence + tiny network = INCONCLUSIVE
  - 艮: No early evidence + **LARGE productive network** = Can still analyze through coherence!

### Initial Observations from Seal Script

**Seal script form (S05942):**
- Visual structure shows 目 (eye) + person-like element looking backwards
- Appears to depict eye + person turning back/looking away
- Traditional decomposition: Component 艮 (name: person-look-away 艮 gěn)
- From eye 目 mù and person-right 匕 bǐ

**Shuowen explanation:**
- 很也從卜主目匕互不相見
- "Meaning blunt/obstinate" (本义 = blunt, stubborn)
- Structure: From 目 (eye) + 匕 (person) - mutually not seeing each other
- Eyes not meeting, looking away from each other

**I Ching (八卦) association:**
- ☶ Gèn trigram = Mountain (山)
- Represents: Stillness, stopping, boundary, limit
- Direction: Northeast
- Quality: Stability, immobility, firmness

**User correction: "艮 is related to eye/sight" (NOT related to 良 which is about seed growth)**

**CRITICAL INSIGHT:**
- 艮 contains 目 (eye) as structural component
- Shuowen: 目 + 匕 = "eyes not meeting, looking away from each other"
- This suggests **VISUAL/SIGHT-RELATED** core semantic, NOT firmness/foundation

**Revised hypothesis to test:**
1. **Primary hypothesis: EYE/SIGHT/VISION-RELATED** - looking, gazing, visual perception
2. How does "looking away" or "eye" relate to the character network?
3. Can visual/sight semantics explain 根, 跟, 很, 限, 眼 etc.?
