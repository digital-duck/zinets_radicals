## Heavenly Stems and Earthly Branches

- 
天干地支简称“干支”，是中国古代天文历法的基本计数单位，源自中国远古时代对天象的观测。“干”原指树干，“支”原指树枝。
天干有十个，依次为：甲、乙、丙、丁、戊、己、庚、辛、壬、癸；
地支有十二个，依次为： 子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥。

十天干和十二地支按固定的顺序相互配合，组成干支纪元法。

中国民间又将十二地支与十二生肖（鼠、牛、虎、兔、龙、蛇、马、羊、猴、鸡、狗、猪）相配，与农历年相对应，如壬寅年为虎年，癸卯年为兔年。

干支纪年法以六十年为一个周期，称为一“甲子”。
这种纪年法反映了中国古人循环往复、生生不息的历史观。

此外，天干地支还广泛应用于术数、计数、命名等领域。
