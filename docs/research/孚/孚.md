**孚 = 爪 (hand/grasp) + 子 (child) = PROTECTIVE HOLDING**

**Core meaning:** Hand securely holding/cradling child - protective grasp of offspring

**Components:**
- **爪** (zhǎo) - hand, grasp, claw
- **子** (zǐ) - child, baby, offspring

**Visual depiction:** Parent hand securely grasping/holding child protectively

**This is the FOUNDATION** - all other characters build upon this core "protective holding" meaning:
- Physical holding (俘 captive, 脬 bladder)
- Protective enclosing (稃 husk, 莩 membrane, 郛 wall)
- Nurturing holding (乳 nursing, 孵 hatching)
- Supportive holding (浮 float - water holds up)
- Abstract extension (孚 trust - reliable holding → trustworthy)
